Отчёт по лабораторной работе №1и Практическому заданию 01
В ходе работы я освоила базовые навыки работы с VIM, управления процессами и systemd, а также научилась парсить nginx-логи.

Работа в VIM. Я научилась открывать, редактировать и сохранять файлы через редактор VIM: открывала файл (vim /path/to/file), переключалась в режим ввода (i), выходила из него (ESC), сохраняла и выходила (:wq), выходила без сохранения (:q!), включала нумерацию строк (:set nu). Путалась в режимах, но быстро привыкла.

Управление службами (systemd). Я научилась писать свой systemd-сервис, который автоматически перезапускается при падении. Получила навыки в написание bash-скриптов, в данном случае my_script.sh с бесконечным циклом, создала unit-файл /etc/systemd/system/my_service.service. Проверила работу скрипта с помощью перезапуска через убийство процесса (kill -9). Поняла, почему systemctl stop не перезапускает сервис — Restart=always работает только при падении, а не при ручной остановке.

Парсинг логов. Я собрала конвейер из утилит для анализа nginx-лога и получения топ-10 IP по числу запросов: awk '{print $1}' /var/log/nginx/access.log | sort | uniq -c | sort -nr | head -10. Для JSON-лога использовала jq -r '.remote_addr'
